## What You'll See

- **Auth flow** — "Try Demo" instant login with JWT
- **Chess** — Full plugin lifecycle: activation → board render → moves → mid-game AI analysis → context retention
- **Weather** — Server-side API proxy, no user auth, simple query-response pattern
- **Spotify** — OAuth2 popup flow, secure token storage, third-party API integration
- **App switching** — Seamlessly switch between apps in a single conversation
- **Admin panel** — App review with approve/reject workflow for K-12 trust & safety
- **Architecture** — WebSocket + postMessage dual-channel design, tool call round-trip explained
